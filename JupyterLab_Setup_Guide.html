What is JupyterLab?

JupyterLab is an interactive development environment for working with Python notebooks.

Why JupyterLab?

  • Interactive: Run code cell by cell
  • Visual: See results immediately
  • Educational: Perfect for learning
  • Documentation: Combine code, text, and visualizations

Think of it as: A notebook where you can write code, see results, and take notes all in one place!

Option 1: JupyterLab Online

Using jupyter.org/try (Recommended for Beginners)

  1. 1Go to jupyter.org/try

Note: This is an experimental demo. For persistent work, consider local installation.

Option 2: Local Installation

Install JupyterLab on Your Computer

Step 1: Install Python

  1. Download Python 3.7+ from python.org
  2. During installation, check "Add Python to PATH"
  3. Complete the installation

Step 2: Install JupyterLab

Open Terminal (Mac/Linux) or Command Prompt (Windows) and run:

pip install jupyterlab

Step 3: Launch JupyterLab

jupyter lab

This will open JupyterLab in your web browser automatically at http://localhost:8888

Advantages: Full control, persistent files, works offline, better for long-term projects

Setting Up Folder Paths

For Local JupyterLab

Option 1: Relative Path (Recommended)

# If your notebook and data are in the same folder
df = pd.read_csv("sensor_data.csv")

# If data is in a subfolder
df = pd.read_csv("data/sensor_data.csv")

Option 2: Absolute Path

# Windows example
df = pd.read_csv("C:/Users/YourName/data/sensor_data.csv")

# Mac/Linux example
df = pd.read_csv("/Users/YourName/data/sensor_data.csv")

Best Practice: Use relative paths - easier to share and move projects

Installing Required Libraries

What You Need

For this course, you'll need:

  • pandas - Data manipulation
  • numpy - Numerical computing
  • matplotlib - Basic plotting
  • seaborn - Statistical visualizations

Installing Libraries

In JupyterLab Online (jupyter.org/try)

Try importing first - some libraries may be pre-installed:

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

If a library is missing, install it:

# Install in JupyterLab
!pip install pandas numpy matplotlib seaborn

Note: The ! runs shell commands in the notebook

Note: Installations in the online demo are temporary. For persistent setup, use local installation.

Installing Libraries

In Local JupyterLab

Method 1: Using pip in Terminal

Open Terminal/Command Prompt and run:

pip install pandas numpy matplotlib seaborn

Method 2: Install in Notebook

Run this in a code cell:

import sys
!{sys.executable} -m pip install pandas numpy matplotlib seaborn

Note: Restart the kernel after installing libraries

JupyterLab Basics

How to Use JupyterLab

Creating a New Notebook:

  • Click "File" ‚Üí "New" ‚Üí "Notebook"
  • Or click the "+" button in the launcher

Running Code:

  • Click in a code cell
  • Type your code
  • Press Shift + Enter to run
  • Or click the "Run" button

Cell Types:

  • Code: For Python code
  • Markdown: For text and documentation

Quick Tips

JupyterLab Shortcuts

  • Shift + Enter - Run cell and move to next
  • Ctrl + Enter - Run cell (stay in place)
  • A - Insert cell above
  • B - Insert cell below
  • DD - Delete cell (press D twice)
  • M - Change to Markdown cell
  • Y - Change to Code cell

Pro Tip: Use Markdown cells to add notes and explanations between code cells!
